Announces the general availability of SonarQube CLI, a tool enabling AI agents to perform secrets detection, code quality scanning, dependency risk analysis, and issue querying through a single command-line interface. The CLI supports the Guide-Verify-Solve framework and integrates with Claude Code and GitHub Copilot through hooks and MCP connections.
