A controlled study found that AI agents working on cleaner codebases used 7.1% fewer input tokens, 8.5% fewer output tokens, and revisited files 34% less often. Structural quality directly impacts token consumption, and enforcing cognitive complexity limits and clean module boundaries yields cost savings across repeated agent sessions.
