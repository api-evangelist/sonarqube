AI-generated Terraform introduces security risks through four recurring failure patterns: permissive defaults, missing security blocks, hardcoded values, and stale provider patterns. Peer-reviewed benchmarks show frontier LLMs pass real-world Terraform tasks below 40%, and traditional validation tools miss these issues because they verify syntax rather than security posture. SonarQube provides coverage from authoring through merge-gate analysis.
